# [环路检测](https://leetcode-cn.com/problems/linked-list-cycle-lcci/)

## 思路

1. 快慢指针可以检测环，但是无法给出环的开头节点，我开始想到的就是在链表里加个属性，visited。当第一次出现 visited 是 true，就表示找到了环的开始节点

```js
var detectCycle = function (head) {
  var p = head
  var index = 0
  while (p) {
    if (!p.visited) {
      p.visited = true
    } else {
      //已经访问过
      return p
    }
    index++
    p = p.next
  }
  return p
}
```
