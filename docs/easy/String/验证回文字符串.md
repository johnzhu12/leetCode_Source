# [回文字符串](https://leetcode-cn.com/explore/interview/card/top-interview-questions-easy/5/strings/36/)

```js
var isPalindrome = function (s) {
    s = s.toLowerCase()
    s = s.replace(/[^a-z0-9]/g, "")
    if (s.split("").reverse().join("") == s) {
        return true;
    } else {
        return false;
    }
};
```
